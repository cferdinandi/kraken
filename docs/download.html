Kraken

Download

There are three ways to use Kraken on your project:

  • Download Kraken directly from GitHub.
  • Clone Kraken from GitHub: git@github.com:cferdinandi/kraken.git
  • Install Kraken using your favorite package manager:
    • NPM: npm install cferdinandi/kraken
    • Bower: bower install https://github.com/cferdinandi/kraken.git
    • Component: component install cferdinandi/kraken
