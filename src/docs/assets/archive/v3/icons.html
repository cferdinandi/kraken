Icons

Kraken includes a lightweight feature test for @font-face and pseudo selector support. The Sass version also includes a placeholder file for working with icon fonts.

Feature Test

When a browser supports @font-face and pseudo selectors, the feature-test.js file adds the .font-face class to the <html> element. You can hook into this class in your CSS to display fallback text and avoid weird characters on non-supporting browsers.

Icon Font Starter

The Sass version of Kraken also includes a placeholder file, _icons.scss, for working with icon fonts. This file includes:

  • A Sass mixin for including fonts with @font-face.
  • Starter classes—with .font-face prefix—for including icons in your markup.
  • A class for fallback text for non-supporting browsers and screenreaders.
